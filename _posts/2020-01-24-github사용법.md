깃허브(GitHub)란 깃을 사용하는 프로젝트를 지원하는 웹호스팅 서비스이다.  
Git과 GitHub는 별개이며 Git을 통해 프로젝트를 진행하면서 여러사람과 저장소를 공유할 수 있는데,
이때 프로젝트를 공유할 수 있는 네트워크상의 저장공간이 필요하고 이러한 네트워크상의(인터넷) 저장공간을 제공해 주는 서비스 중 하나가 GitHub이다.

##1.## Github에 저장소 작성(git init) 또는 복제(git clone)  
##2.## 파일 작성 또는 편집  
##3.## 파일을 git인덱스에 추가(add)  
##4.## 결과를 로컬 저장소에 커밋(commit)  
##5.## 로컬 저장소를 푸쉬해 원격 저장소에 반영(push)  

##Git Bash Here##을 클릭한 후, Bash창이 나오면  
git add .  
git commit -m "파일설명"  
git push -u origin master
